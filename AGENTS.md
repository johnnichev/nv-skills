# nv:context

Context engineering setup for AI coding agents. Works with Claude Code, Cursor, GitHub Copilot, Windsurf, Aider, and Gemini CLI.

## Skill

- `nv-context` — Auto-detects tools, analyzes codebase, finds landmines, generates multi-level configs, hooks, session management, token budgets. 200+ research sources.

## Install

```bash
npx skills add johnnichev/nv-context -g -y
```
